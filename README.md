# Instructions

## Global dependencies
	- node/npm
    - bower
    - gulp
    - http-server (optional)

## Build instructions
    - cd <project root>
    - npm install
    - bower install
    - gulp

## Run locally
    - cd <project root>
    - http-server dist (or any other server)
    - open browser with server's host
